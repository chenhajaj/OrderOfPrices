# Order of Prices — Online Experiment Platform

Web application used to run the human-subjects experiment for the paper:

**[Ordering Effects and Belief Adjustment in the Use of Comparison Shopping Agents](https://ojs.aaai.org/index.php/AAAI/article/view/8837)** — Chen Hajaj, Noam Hazon, and David Sarne, *AAAI 2014*.

Participants acted as buyers on a simulated comparison shopping site, choosing among sellers listed at different prices. The experiment studies how the order in which prices are presented affects purchasing decisions and belief updating — providing empirical grounding for strategic ordering in comparison shopping agents.

Built as an ASP.NET web application designed for deployment as an Amazon Mechanical Turk HIT.
